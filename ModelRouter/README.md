
Problem : The Cost and quality with multple LLMs create challenges in multi model environment to use the right model for right task. 
Model Router : The model router is designed to address the model routing problem based on multiple categorization of the user task query. The Model Router implements a simulation for developers to try the Model Router engine, 
test with real LLM models with real life queries and modify to adopt for the real life agentic AI applicaitons.
